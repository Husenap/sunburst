# Sunburst

An OpenGL Renderer written in Scala3.
Uses:
* LWJGL
* GLFW
* OpenGL
* ImGui

### Usage

You can:
* compile code with `sbt compile`
* run it with `sbt app/run`
* assembly it with `sbt assembly`
